# React Application: Random Number Generator
* Learn in this project to generate a random number using State Hook and the onClick event. Learn in this project how to generate a random number using State Hook and the onClick event. *

What is State Hook?
The State Hook (useState), basically allows us to create a variable that, when changing the value, will automatically be changed on the user's screen.


What is the onClick event?
When clicking on a button it is common for something to happen. To call this action we use the onClick event.

In this example, when the user clicks the button, the code associated with the onClick event will be executed (the generation of the random number).
